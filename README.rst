Terraform-Iot-Air-Pollution
============================

Requirements
------------

This is a list of applications that need to be installed previously to
enjoy all the goodies of this configuration:

-  `Pyenv`_
-  `Docker`_
-  `Terraform`_
-  `Terragrunt`_

Quick Start
===========

Environment
-----------

.. code-block:: bash

    $ make environment

Dependences
-----------

.. code-block:: bash

    $ make setup
